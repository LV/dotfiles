#+title: DevX instructions

To setup a DevX environment, do the following:
- Under *User Details*
  - *Repository*: /GitHub/ | =lv/dotfiles=
  - *Target Path*: =~/.config=
  - *Install Command*: /leave blank/

Once the repository is created and you SSH into the environment, run the following command:

#+begin_src sh
source /root/.config/scripts/bash/devx.sh
#+end_src
